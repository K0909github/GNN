# 最長増加部分列

- 出典: GOC
- 問題ID: 9
- カテゴリ: 動的計画法
- 難易度: Medium
- サブタイトル: LeetCode 300

## 問題文
整数配列numsが与えられる。厳密に増加する最長部分列の長さを返せ（連続でなくてよい）。

## 例
```text
入力: [10,9,2,5,3,7,101,18]
出力: 4  # [2,3,7,101]
```

## ヒント
dp[i] = nums[i]で終わる最長増加部分列の長さ。dp[i] = max(dp[j]+1) for j<i where nums[j]<nums[i]。

## 計算量
時間: O(N²)　空間: O(N)　※O(N log N)解法もあり
