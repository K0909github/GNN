## ???
依存グラフが与えられる。各ノードはプロジェクト、有向エッジは「依存関係」を表す。
サイクルに含まれるプロジェクト（循環依存で永遠に完了できないもの）を全て返せ。

## ?
```text
入力: n=4, edges=[[0,1],[1,2],[2,0],[3,1]]
出力: [0, 1, 2]  # 3はサイクル外
```

## ???
トポロジカルソート（Kahn's algorithm）でサイクル検出。in-degreeが0のノードをキューに入れ、除去できなかったノードがサイクル。
