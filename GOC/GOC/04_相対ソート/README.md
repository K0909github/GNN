# 相対ソート

- ??: GOC

## ???
配列arr1とarr2が与えられる。arr2の要素は全てユニークでarr1に含まれる。
arr1をarr2の順番に従ってソートせよ。arr2にない要素は末尾に昇順で置く。

## ?
```text
入力: arr1=[2,3,1,3,2,4,6,7,9,2,19], arr2=[2,1,4,3,9,6]
出力: [2,2,2,1,4,3,3,9,6,7,19]
```

## ???
arr2のインデックスをキーにした辞書を作る。arr1をカスタムキー（arr2にあれば順位、なければ(大きい数, 値)）でソート。

## ???
時間: O(N log N)　空間: O(N)
