# アナグラムのグループ化

- 出典: GOC
- 問題ID: 5
- カテゴリ: ソート＋ハッシュ
- 難易度: Medium
- サブタイトル: LeetCode 49

## 問題文
文字列のリストが与えられる。アナグラム同士をグループ化して返せ。

## 例
```text
入力: ["eat","tea","tan","ate","nat","bat"]
出力: [["bat"],["nat","tan"],["ate","eat","tea"]]
```

## ヒント
各文字列をソートしたものをキーとしてハッシュマップに入れる。sorted('eat') == sorted('tea') == 'aet'。

## 計算量
時間: O(N·K log K)　空間: O(N·K)　※K=最長文字列長
