# 回文判定

- 出典: PKSHA
- 問題ID: 1-1
- レベル: A
- 目安時間: 〜10分
- タグ: 文字列

## 問題文
文字列 S が与えられます。S を逆順にした文字列と S が一致する場合は "Yes"、そうでない場合は "No" を出力してください。

## 制約
- 1 ≤ |S| ≤ 10⁵
- S は英小文字のみからなる

## 入出力例
### 例1
```text
入力:
racecar

出力:
Yes
```
注釈: 逆から読んでも同じ

### 例2
```text
入力:
hello

出力:
No
```
注釈: 逆にすると 'olleh'

### 例3
```text
入力:
a

出力:
Yes
```
注釈: 長さ1は常に回文

## ヒント
Python なら s == s[::-1] で一発です。

## 解説
文字列を逆順にして元の文字列と比較するだけです。

【Python 例】
S = input()
print("Yes" if S == S[::-1] else "No")

計算量: O(|S|)
