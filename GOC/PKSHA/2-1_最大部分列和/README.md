# 最大部分列和

- 出典: PKSHA
- 問題ID: 2-1
- レベル: A
- 目安時間: 15〜20分
- タグ: 動的計画法, Kadane

## 問題文
N 個の整数からなる数列 a₁, a₂, ..., aₙ が与えられます。空でない連続部分列の和の最大値を出力してください。

## 制約
- 1 ≤ N ≤ 2×10⁵
- −10⁹ ≤ aᵢ ≤ 10⁹

## 入出力例
### 例1
```text
入力:
9
-2 1 -3 4 -1 2 1 -5 4

出力:
6
```
注釈: [4, -1, 2, 1] の和が最大

### 例2
```text
入力:
1
-5

出力:
-5
```
注釈: 要素が1つなので選ぶしかない

### 例3
```text
入力:
4
1 2 3 4

出力:
10
```
注釈: 全要素の和

## ヒント
Kadane のアルゴリズム：現在の連続和が負になったらリセットするとどうなりますか？

## 解説
Kadane's Algorithm の典型です。

【Python 例】
N = int(input())
A = list(map(int, input().split()))
cur = best = A[0]
for a in A[1:]:
    cur = max(a, cur + a)
    best = max(best, cur)
print(best)

直感: cur が負なら捨てて次の要素から再スタート
計算量: O(N)
