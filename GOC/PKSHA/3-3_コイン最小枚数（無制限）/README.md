## ???
dp[j] = 金額 j を作る最小コイン枚数 と定義します。dp[0] = 0 から始めてみましょう。

## ??
完全ナップサック問題（無制限コイン）のDPです。

【Python 例】
N, M = map(int, input().split())
C = list(map(int, input().split()))
INF = float('inf')
dp = [INF] * (M + 1)
dp[0] = 0
for j in range(1, M + 1):
    for c in C:
        if j >= c and dp[j - c] != INF:
            dp[j] = min(dp[j], dp[j - c] + 1)
print(dp[M] if dp[M] != INF else -1)

計算量: O(N×M)
※ 遷移: 金額 j を作るには「j-c を作って c を1枚足す」
