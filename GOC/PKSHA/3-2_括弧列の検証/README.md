# 括弧列の検証

- 出典: PKSHA
- 問題ID: 3-2
- レベル: B
- 目安時間: 20〜25分
- タグ: スタック, 文字列

## 問題文
'('、')'、'['、']'、'{'、'}' のみからなる文字列 S が与えられます。S が正しい括弧の対応になっているかを判定し、"Yes" または "No" を出力してください。正しい括弧列とは、全ての開き括弧に対応する閉じ括弧が正しい順序で存在するものです。

## 制約
- 1 ≤ |S| ≤ 10⁵
- S は '('、')'、'['、']'、'{'、'}' のみ

## 入出力例
### 例1
```text
入力:
()[]{}

出力:
Yes
```

### 例2
```text
入力:
([{}])

出力:
Yes
```

### 例3
```text
入力:
(]

出力:
No
```
注釈: 対応が違う

### 例4
```text
入力:
([)]

出力:
No
```
注釈: 入れ子が崩れている

### 例5
```text
入力:
{[]}

出力:
Yes
```

## ヒント
スタックを使いましょう。開き括弧はスタックに積み、閉じ括弧はスタックトップと照合します。

## 解説
スタックを使う典型問題です。

【Python 例】
S = input()
stack = []
pair = {')': '(', ']': '[', '}': '{'}
for c in S:
    if c in '([{':
        stack.append(c)
    else:
        if not stack or stack[-1] != pair[c]:
            print("No")
            exit()
        stack.pop()
print("Yes" if not stack else "No")

計算量: O(|S|)
