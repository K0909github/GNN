# 予算内で最多購入

- 出典: PKSHA
- 問題ID: 1-2
- レベル: B
- 目安時間: 15〜20分
- タグ: 貪欲法, ソート

## 問題文
N 個の商品があり、i 番目の商品の値段は p_i 円です。予算 M 円の中でできるだけ多くの商品を購入するとき、最大購入個数を出力してください（各商品は 1 個しか買えません）。

## 制約
- 1 ≤ N ≤ 2×10⁵
- 1 ≤ M ≤ 10⁹
- 1 ≤ p_i ≤ 10⁹

## 入出力例
### 例1
```text
入力:
5 1000
100 200 300 400 500

出力:
4
```
注釈: 100+200+300+400=1000

### 例2
```text
入力:
3 100
200 300 400

出力:
0
```
注釈: 最安でも200円で予算オーバー

### 例3
```text
入力:
4 10
1 2 3 4

出力:
4
```
注釈: 1+2+3+4=10 ぴったり

## ヒント
安い商品から順番に買っていくとどうなるでしょうか？

## 解説
値段を昇順ソートし、累積和が M を超えるまで買い続けます。

【Python 例】
N, M = map(int, input().split())
P = sorted(map(int, input().split()))
count = total = 0
for p in P:
    if total + p <= M:
        total += p
        count += 1
    else:
        break
print(count)

計算量: O(N log N)
