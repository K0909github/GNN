## ???
N 個のタスクがあり、i 番目のタスクは時刻 sᵢ に開始し時刻 eᵢ に終了します（同じ時刻に終了・開始するタスクは重複なく連続して実行できます）。1 人の作業者が重複なく実行できるタスクの最大個数を出力してください。

## ???
終了時刻が早いタスクから貪欲に選ぶのが最適解です。

## ??
終了時刻でソートして貪欲に選択します。

【Python 例】
N = int(input())
tasks = [tuple(map(int, input().split())) for _ in range(N)]
tasks.sort(key=lambda x: x[1])  # 終了時刻でソート
count = 0
last_end = -1
for s, e in tasks:
    if s >= last_end:
        count += 1
        last_end = e
print(count)

計算量: O(N log N)
※ 証明: 終了が早いタスクを選ぶと「残り時間」が最大化される
