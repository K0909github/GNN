## ???
N が最大 10¹² なので全探索は無理です。√N まで試し割りして、i が約数なら N/i も約数であることを使います。

## ??
i=1 から √N まで試し割りします。

【Python 例】
import math
N = int(input())
count = 0
i = 1
while i * i <= N:
    if N % i == 0:
        count += 2 if i * i != N else 1
    i += 1
print(count)

計算量: O(√N) ≈ O(10⁶) で間に合います
