# 約数の個数

- 出典: PKSHA
- 問題ID: 3-1
- レベル: A
- 目安時間: 10〜15分
- タグ: 数学, 整数論

## 問題文
整数 N が与えられます。N の約数の個数を出力してください。

## 制約
- 1 ≤ N ≤ 10¹²

## 入出力例
### 例1
```text
入力:
12

出力:
6
```
注釈: 約数は 1,2,3,4,6,12

### 例2
```text
入力:
1

出力:
1
```
注釈: 約数は 1 のみ

### 例3
```text
入力:
1000000007

出力:
2
```
注釈: 素数なので約数は 1 と自身のみ

## ヒント
N が最大 10¹² なので全探索は無理です。√N まで試し割りして、i が約数なら N/i も約数であることを使います。

## 解説
i=1 から √N まで試し割りします。

【Python 例】
import math
N = int(input())
count = 0
i = 1
while i * i <= N:
    if N % i == 0:
        count += 2 if i * i != N else 1
    i += 1
print(count)

計算量: O(√N) ≈ O(10⁶) で間に合います
